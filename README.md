## Description
SingKey is a web-based tool that generates an ultra-secure password 
based on your hand movements. The password is automatically copied 
to your clipboard — no need to type or remember it.

The program detects:
- The number of fingers you raise
- The combination of fingers used (e.g. index + thumb)
- A closed fist, which is also recognized as a gesture

To retrieve your password, simply reproduce the same hand gestures.
SingKey could also potentially be used for password recovery.

---

## How to use
1. Go to the website
2. When prompted, allow the following permissions:
   -  **Camera** — to detect your hand gestures
   -  **Motion sensor** — to track your movements
   -  **Clipboard** — to automatically copy your password
3. Click on the "Genrate a password" button
4. Perform your hand gestures in front of the camera
5. Your password is automatically generated and copied
   to your clipboard
6. To retrieve it later, visit the site and reproduce 
   the same gestures

⚠️ **These permissions are required for SingKey to work properly.**
No installation required. Works directly in your browser.
